# SongList Studio

SongList Studio 是一个面向音乐主播、驻唱和 KTV 高频用户的个人歌曲库与歌单生成应用。

核心目标：

- 管理用户自己会唱的歌曲。
- 根据场景和分类维度生成歌单。
- 支持简洁背景设置。
- 导出可发布的歌单图片。
- 管理历史发布过的歌单。

## 技术栈

- React
- TypeScript
- Vite
- npm

## 启动

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
